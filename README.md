# Applied Machine Learning (AML) Lab

Welcome to the AML Lab repository. This project contains various machine learning experiments and datasets for learning and practice.

## Project Structure

- **`experiments/`**: Contains Jupyter notebooks (`.ipynb`) and Python scripts (`.py`) for the different lab experiments (Exp 1 to 12).
- **`datasets/`**: Data files used for training and evaluating the models in the experiments.


## Getting Started

1. Navigate to the `experiments/` directory.
2. Open the Jupyter Notebook or Python script for the desired experiment.
3. Ensure you have the necessary datasets available in the `datasets/` directory before running the code.

## Prerequisites

To run these experiments, you'll generally need:
- Python 3.x
- Jupyter Notebook
- Standard ML libraries (e.g., scikit-learn, pandas, numpy, matplotlib)
